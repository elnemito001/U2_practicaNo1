Comunicación entre Componentes

¿Qué verás aquí?

Este ejemplo demuestra cómo un componente PADRE puede enviar datos a un componente HIJO usando el decorador @Input() de Angular.

El flujo de datos es: Padre → Hijo (unidireccional)

📝 Conceptos Clave

  • @Input(): Permite recibir datos desde el componente padre
  • [propiedad]="valor": Sintaxis de property binding en Angular
  • Data Flow: Los datos fluyen de arriba hacia abajo en el árbol de componentes
